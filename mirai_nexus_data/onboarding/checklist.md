# Employee Onboarding Checklist (Mirai Nexus)

- **Offer & Documentation**  
  - Offer letter signed and returned  
  - Government ID and tax forms submitted  
  - NDA and other compliance documents signed  

- **Account Provisioning**  
  - Company email account created  
  - Slack and other internal communication tools access granted  
  - GitHub / code repository access provisioned  
  - VPN and network credentials issued  

- **Equipment & Workspace**  
  - Laptop, monitor, docking station, and peripherals shipped  
  - Software licenses and productivity tools installed  
  - Desk setup (for in-office employees) or remote workspace guidelines provided  

- **Training & Orientation**  
  - Orientation session scheduled  
  - HR policies and handbook reviewed  
  - Security and IT onboarding completed  

- **Miscellaneous**  
  - Team introduction scheduled  
  - Initial 30/60/90 day goals shared with manager
